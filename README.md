# snake

```
usage: snake.py [-h] [-b]

Eat food, stay alive, get the high score!

optional arguments:
  -h, --help        show this help message and exit
  -b, --borderless  enable wrap-around at arena borders

Eat food to grow or shrink the snake.
Gain points for survival.
Don't hit the borders or eat yourself!

Food Types:
- White: Grow by 1
- Green: Grow by 3
- Gold:  Grow by 9
- Red:   Shrink by 3
```
